# IBM Quantum Software Developer Certification Study Group

This repository contains **resources, study materials, practice repositories, and discussion links** for participants preparing for the  
**IBM Certified Quantum Computation using Qiskit v2.x Developer Associate** certification.

The goal of this repository is to organize all useful materials in one place for the study group.
